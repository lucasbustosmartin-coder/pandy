# SQL – Pandi

En esta carpeta van todos los scripts SQL para Supabase: tablas, RLS, migraciones, datos.

- Nomenclatura sugerida: `supabase_*.sql` para objetos de Supabase.
- Ejecutar en el **SQL Editor** del proyecto Supabase en el orden que indique la documentación o los comentarios de cada archivo.

Cuando definas el modelo de datos de la app, creá aquí los DDL correspondientes.
